> Vagalume API Node Module


## Install

```sh
$ npm install --save node-vagalume
```


## Usage

```js
var nodeVagalume = require('node-vagalume');

nodeVagalume('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
